# feedcache

Mirrors public internet top-list / metadata feeds into git-backed data repositories:

- Cisco Umbrella Top 1M → [umbrella-top1m-cache](https://github.com/wangmm001/umbrella-top1m-cache)
- Tranco Top 1M → [tranco-top1m-cache](https://github.com/wangmm001/tranco-top1m-cache)
- Cloudflare Radar rank buckets → [cloudflare-radar-rankings-cache](https://github.com/wangmm001/cloudflare-radar-rankings-cache)

See `docs/design.md` for design details.
